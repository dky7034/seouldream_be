# 임원 대시보드: 출석 통계 설계 명세

1.  **개요**
    임원단(EXECUTIVE)이 교회 전체의 출석 흐름을 직관적으로 파악할 수 있도록, 대시보드에는 최근 3개월 출석률 추이를 기본으로 제공한다.
    출석 통계 데이터는 전체 교회(Member 전체)를 기준으로 집계한다.

2.  **기본 화면: 최근 3개월 출석률 추이**
    *   대시보드 진입 시 기본값은 최근 3개월 출석률 추이 그래프이다.
    *   최근 3개월(약 12주)은 전체 교회의 “현재 분위기(Trend)”를 파악하기 적합하다.
    *   이를 통해 임원단은 다음과 같은 판단이 가능하다.
        *   최근 출석이 증가하고 있는가?
        *   감소 추세는 아닌가?
        *   단기간 급격한 변화가 있는가?

3.  **기간 선택 옵션 (필터)**
    대시보드 화면에서는 기본값(최근 3개월) 외에 다른 기간을 선택할 수 있도록 한다.

    **제공 옵션:**
    *   최근 1개월
    *   최근 3개월 (기본값)
    *   최근 6개월
    *   최근 12개월 또는 올해 전체

    **목적:**
    *   대시보드에서 “요즘 흐름(단기 Trend)”을 바로 확인
    *   필요 시 상세 분석을 위해 기간을 확장하여 확인
    *   계절성, 연간 변화, 장기 추이를 유연하게 분석 가능

4.  **그래프 + 핵심 지표 조합 제공**
    출석률 그래프와 함께 다음 3개의 핵심 요약 지표를 제공한다.
    1.  이번 주 출석률
    2.  최근 3개월 평균 출석률
    3.  작년 같은 기간 출석률(YoY 비교)

    **목적:**
    *   단순 그래프만으로 파악하기 어려운 “현재 상태 + 중기 추세 + 계절성”을 한 번에 전달
    *   임원단이 빠르게 전체 상황을 판단할 수 있도록 지원

5.  **대시보드 vs 상세 페이지 역할 분리**

    *   **대시보드(Dashboard) 역할**
        *   빠르고 직관적인 주요 지표 요약 제공
        *   최근 3개월 출석률 추이(기본값)
        *   단순한 YoY(전년 대비) 또는 전월 대비 변화량 제공

    *   **상세 통계 페이지(Reports / Analytics) 역할**
        *   분석 목적의 전용 화면
        *   1년 또는 전체 기간 출석률 분석
        *   부서별 출석률 비교
        *   셀별 출석률 비교
        *   연속 결석자 분석
        *   커스텀 기간 선택
        *   심층 표/그래프 및 상세 데이터 제공

    **목적:**
    *   대시보드: “한눈에 보는 요약판” → 빠른 판단
    *   상세 페이지: “운영 전략용” → 깊은 분석

6.  **결론**
    본 설계 방식은 다음 장점을 가진다.
    *   직관적이며 사용성이 높음
    *   확장성 우수(현황 → 상세 분석 구조 유지)
    *   임원단이 실제로 필요로 하는 ‘공동체 전체 흐름’ 파악에 최적화
    *   대시보드의 복잡도 증가를 방지하며, 상세 페이지에서 충분한 분석 기능 제공 가능
